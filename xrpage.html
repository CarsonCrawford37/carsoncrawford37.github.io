XR Development Projects

Alzheimer's Pre-Diagnostic Test

Implementing smell into virtual reality, this pre-test is meant to locate any indicators of early-onset dementia. My job was to ensure the device worked properly with the program as well as making the test much more accessible to older users.

Learn more

Memory City

Utilizing the immersive visual elements of VR alongside a custom-made scent device, Memory City aimed to explore the impact of scent on spatial learning and memory recollection. As the project lead, my responsibilities included organizing tasks for artists, programmers, and UX designers, as well as maintaining clear communication between the client and our team.

Learn more

Magi VR

Magi VR is a puzzle, spell-casting, dungeon crawler that implements voice recognition for a unique and mysterious experience. This was a solo development project to further expand my knowledge of VR and game development.

Learn more

Printer PARtner

With accessibility as our top priority, we developed this AR app to simplify the use of printers in the Boise State Albertson's Library with step-by-step instructions. My role involved integrating Siri's text-to-speech and haptic feedback for enhanced accessibility, along with incorporating image anchoring and sequencing events.

Learn more

Memory Scent Simulation

Building on the technology from Memory City and the Pre-Diagnostic test, this simulation aims to explore how the complexity of scent tones influences memory. My responsibilities included designing a tutorial room to guide players through object selection controls and optimizing the UI for improved readability by proctors.

Learn more

AR Mini Golf Game

Using Unity 3D, I and three others made a mini golf AR game with image anchoring, directional input, and multiple maps. As the main programmer, my job was to implement the main meat of the game such as the anchoring, directional input, and win condition.

Learn more

VR Driving Simulation

For my high school senior project, I decided to create a VR driving simulation that was aimed at helping new student drivers on the road become more vigilant and skilled before getting behind an actual wheel. This was a solo project that was entirely self-taught and was the first project that propelled me into the field of XR.
